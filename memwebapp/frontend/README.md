# Memo App Dashboard

A modern, customizable dashboard platform for visualizing customer conversations and analytics using AI.

---

## ğŸš€ Features

- **Custom Dashboards:** Create, edit, and manage multiple dashboards.
- **AI-Powered Insights:** Generate charts and word clouds from conversation data.
- **Interactive Visualizations:** Bar, Pie, Line, Area, Scatter, and Word Cloud charts.
- **Bookmarking:** Save and filter your favorite charts.
- **Responsive UI:** Built with React, Tailwind CSS, and shadcn-ui.
- **Popup & Fullscreen Views:** Expand charts for detailed analysis.
- **Data Filtering:** Search and filter charts by query or bookmark status.
- **Testing:** Includes unit and integration tests for components and utilities.
